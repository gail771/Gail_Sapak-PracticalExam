    What is HTML?
    • HTML stands for Hyper Text Markup Language
    • HTML is used to make web pages, control the display of web pages and content
    • HTML you can insert pictures, music, videos, animations and other multimedia
    • HTML you used to retrieve information via link
    • HTML you can use forms to obtain user information and realize interaction
